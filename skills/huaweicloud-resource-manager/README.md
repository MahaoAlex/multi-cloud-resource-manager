# Huawei Cloud Resource Manager

华为云资源管理器 Claude Code Skill。

## 完整文档

详见项目文档：[华为云完整使用指南](../../docs/huaweicloud-complete-guide.md)

## 快速开始

```bash
# 配置认证
配置华为云认证

# 执行完整扫描
执行完整的华为云资源扫描

# 或使用命令模式
/huaweicloud-scan full_scan --regions=["cn-north-4"] --max_workers=3 --vpc_max_workers=3
```

## 环境变量

| 变量 | 说明 |
|------|------|
| `HWCLOUD_ACCESS_KEY` | Access Key ID |
| `HWCLOUD_SECRET_KEY` | Secret Access Key |
| `HWCLOUD_REGIONS` | 区域列表或 'all' |
